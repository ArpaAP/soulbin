## 서비스 소개

### 서비스 개요

설빈(Soul Bin)은 AI 기반 감정 케어 플랫폼으로, 사용자가 일기 작성 또는 AI 상담 채팅 두 가지 형식 중 하나를 선택해 자신의 심리 상태를 기록하고 분석할 수 있는 서비스입니다. 사용자가 작성한 일기 내용이나 AI와 주고받은 상담 대화를 자연어 처리 AI가 분석하여 현재 감정 상태(예: 불안, 우울, 평온 등)와 성격 경향을 추론합니다. 이러한 분석 결과는 일별 감정 변화 추이, 감정 분포 그래프 등으로 시각화되어 제공되며, 이를 바탕으로 사용자에게 맞춤형 피드백을 제시해줍니다.

현대 사회에서는 '정신병'이라는 단어에 대한 부정적 인식 때문에 자신의 정신 상태를 솔직하게 파악하고 인정하기 어려운 분위기가 있습니다. 사실 많은 정신적 어려움이 뇌와 몸의 화학 작용 등 다양한 요인으로 인해 발생하지만, 사람들은 이를 마음이 약해졌다고 여기며 부정하거나 숨기는 경향이 있습니다. 설빈(Soul Bin)은 이러한 문제를 해결하기 위해, 사용자가 모르는 사이 누적된 감정의 흐름과 패턴을 눈으로 볼 수 있게 시각화하여 심리적 고통을 인지하도록 돕고 그에 대한 작은 해결책을 제공합니다. 이를 통해 사용자는 자신의 스트레스와 감정적 고통을 객관적으로 받아들이고, 이를 해소하기 위한 첫 걸음을 뗄 수 있게 됩니다.

### 타서비스와의 차별점

유사 서비스와의 차별점

**AI 대화형 감정 분석**: 기존의 감정일기 앱들이 사용자가 입력한 감정을 단순 기록하거나 일방적인 코멘트를 제공하는 데 그치는 반면, 설빈은 AI와의 쌍방향 대화를 통해 감정을 분석하고 피드백합니다. 사용자는 일상 대화하듯 고민을 털어놓을 수 있고, AI는 그 내용을 깊이 있게 해석하여 실시간 정서적 공감과 답변을 제공합니다.

**감정 및 성격 시각화 기능**: 일반적인 심리 기록 또는 명상 앱에서는 감정 변화를 시각적으로 표시하는 기능이 미흡한 경우가 많습니다. 설빈은 감정 상태의 변화 추이와 다양한 감정 분포를 그래프로 보여주고, 나아가 사용자의 성격 프로파일까지 시각화하여 제공합니다. 이 직관적인 시각화 덕분에 사용자는 자신의 마음 상태를 한눈에 파악하고 패턴을 이해할 수 있습니다.

**맞춤형 조언과 피드백**: 설빈은 사용자의 현재 감정 상태에 꼭 맞는 응원 문구나 조언을 제공합니다. 예를 들어 우울한 날에는 위로와 함께 간단한 기분 전환 팁을, 불안한 날에는 심호흡 등 안정에 도움이 되는 제안을 건네는 식입니다. 이러한 개인 맞춤 피드백은 주로 정해진 콘텐츠만 제공하는 기존 앱들과 달리, 사용자마다 다른 대응책을 제시한다는 점에서 차별화됩니다.

**두 가지 기록 모드 제공**: 일기 쓰기를 선호하는 사람도 있고 대화로 털어놓는 방식을 편하게 여기는 사람도 있습니다. 설빈은 텍스트 일기 모드와 AI 상담 채팅 모드를 모두 지원하여 사용자 취향에 맞는 표현 방법을 선택하게 합니다. 다른 유사 서비스들이 주로 일기 형식만 제공하거나 챗봇 상담만 가능한 것과 달리, 설빈에서는 필요에 따라 기록 방식을 유연하게 바꿀 수 있어 이용 편의성이 높습니다.

### 구현 내용 및 결과물

### 구현 방식

#### Summary

> Design: **Figma**
> FE/BE: **Next.js 16.0** (with React 19.2 + TypeScript)  
> UI Framework: **Radix UI + Tailwind CSS v4**  
> Database: **PostgreSQL**  
> ORM: **Prisma**  
> Deploy: **Vercel**  
> Etc: **BetterAuth** for authentication, **react-hook-form** for form handling

#### 기술 구현 상세

- React 기반의 풀스택 웹 프레임워크인 Next.js 를 사용하여 서버사이드 렌더링(SSR) 적용
- SSR을 기반으로 Server-side data fetching, image optimization, API Router 등의 기술을 적용
- Auth.js 활용하여 세션 기반의 로그인 구현
- Radix UI 및 Tailwind CSS 활용한 디자인 시스템 구축, 직접 디자인한 커스텀 SVG 아이콘셋 적용
