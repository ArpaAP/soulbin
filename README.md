## 서비스 소개

### 서비스 개요

![서비스 메인 화면](.github/images/01.%20Main%20Screen.png)

설빈(SoulBin)은 AI 기반 감정 케어 플랫폼으로, 사용자가 일기 작성 또는 AI 상담 채팅 두 가지 형식 중 하나를 선택해 자신의 심리 상태를 기록하고 분석할 수 있는 서비스입니다. 사용자가 작성한 일기 내용이나 AI와 주고받은 상담 대화를 자연어 처리 AI가 분석하여 현재 감정 상태(예: 불안, 우울, 평온 등)와 성격 경향을 추론합니다. 이러한 분석 결과는 일별 감정 변화 추이, 감정 분포 그래프 등으로 시각화되어 제공되며, 이를 바탕으로 사용자에게 맞춤형 피드백을 제시해줍니다.

현대 사회에서는 '정신병'이라는 단어에 대한 부정적 인식 때문에 자신의 정신 상태를 솔직하게 파악하고 인정하기 어려운 분위기가 있습니다. 사실 많은 정신적 어려움이 뇌와 몸의 화학 작용 등 다양한 요인으로 인해 발생하지만, 사람들은 이를 마음이 약해졌다고 여기며 부정하거나 숨기는 경향이 있습니다. 설빈(SoulBin)은 이러한 문제를 해결하기 위해, 사용자가 모르는 사이 누적된 감정의 흐름과 패턴을 눈으로 볼 수 있게 시각화하여 심리적 고통을 인지하도록 돕고 그에 대한 작은 해결책을 제공합니다. 이를 통해 사용자는 자신의 스트레스와 감정적 고통을 객관적으로 받아들이고, 이를 해소하기 위한 첫 걸음을 뗄 수 있게 됩니다.

### 타서비스와의 차별점

#### AI 대화형 감정 분석

기존의 감정일기 앱들이 사용자가 입력한 감정을 단순 기록하거나 일방적인 코멘트를 제공하는 데 그치는 반면, 설빈은 AI와의 쌍방향 대화를 통해 감정을 분석하고 피드백합니다. 사용자는 일상 대화하듯 고민을 털어놓을 수 있고, AI는 그 내용을 깊이 있게 해석하여 실시간 정서적 공감과 답변을 제공합니다.

#### 감정 및 성격 시각화 기능

일반적인 심리 기록 또는 명상 앱에서는 감정 변화를 시각적으로 표시하는 기능이 미흡한 경우가 많습니다. 설빈은 감정 상태의 변화 추이와 다양한 감정 분포를 그래프로 보여주고, 나아가 사용자의 성격 프로파일까지 시각화하여 제공합니다. 이 직관적인 시각화 덕분에 사용자는 자신의 마음 상태를 한눈에 파악하고 패턴을 이해할 수 있습니다.

#### 맞춤형 조언과 피드백

설빈은 사용자의 현재 감정 상태에 꼭 맞는 응원 문구나 조언을 제공합니다. 예를 들어 우울한 날에는 위로와 함께 간단한 기분 전환 팁을, 불안한 날에는 심호흡 등 안정에 도움이 되는 제안을 건네는 식입니다. 이러한 개인 맞춤 피드백은 주로 정해진 콘텐츠만 제공하는 기존 앱들과 달리, 사용자마다 다른 대응책을 제시한다는 점에서 차별화됩니다.

#### 두 가지 기록 모드 제공

일기 쓰기를 선호하는 사람도 있고 대화로 털어놓는 방식을 편하게 여기는 사람도 있습니다. 설빈은 텍스트 일기 모드와 AI 상담 채팅 모드를 모두 지원하여 사용자 취향에 맞는 표현 방법을 선택하게 합니다. 다른 유사 서비스들이 주로 일기 형식만 제공하거나 챗봇 상담만 가능한 것과 달리, 설빈에서는 필요에 따라 기록 방식을 유연하게 바꿀 수 있어 이용 편의성이 높습니다.

### 구현 내용 및 결과물

#### 1. 일기 작성 및 AI 감정 분석

![일기 작성 화면](.github/images/03.%20diary-write.png)

- 사용자가 작성한 일기를 OpenAI GPT-4o-mini를 활용하여 자동 분석
- 감정 상태(예: 불안, 기쁨, 슬픔, 분노 등), 강도(1-10 스케일), 관련 태그, 한 줄 요약 추출
- 사용자의 AI 스타일 설정(AUTO, COLD, WARM)에 따른 맞춤형 조언 제공

#### 기술 구현 상세

**프론트엔드**

- **Next.js 16.0** with **React 19.2**: 서버사이드 렌더링(SSR)과 Server Actions를 활용한 풀스택 개발
- **TypeScript**: 타입 안전성을 보장하고 개발 생산성 향상
- **Tailwind CSS v4**: 유틸리티 기반 스타일링으로 일관된 디자인 시스템 구축
- **Radix UI**: 헤드리스 UI 컴포넌트를 기반으로 커스텀 디자인을 구현하여 개발
- **react-hook-form**: 효율적인 폼 상태 관리 및 유효성 검사
- **recharts**: 감정 통계 데이터 시각화를 위한 차트 라이브러리
- **motion (Framer Motion)**: 부드러운 애니메이션 효과
- **date-fns**: 날짜 포맷팅 및 계산

**백엔드**

- **Next.js API Routes**: RESTful API 엔드포인트 구현
- **Server Actions**: React Server Components를 활용한 서버 측 데이터 처리
- **PostgreSQL**: 관계형 데이터베이스로 사용자, 일기, 채팅 데이터 저장
- **Prisma ORM**: 타입 안전한 데이터베이스 쿼리 및 마이그레이션 관리
- **BetterAuth**: 세션 기반 인증 시스템 구현

**AI 및 자연어 처리**

- **LangChain**: AI 워크플로우 구축 및 프롬프트 관리
- **OpenAI API (GPT-4o-mini)**: 감정 분석, 조언 생성, 채팅 응답 생성
- **Structured Output Parser**: JSON 스키마 기반 AI 응답 파싱
- **Zod**: 런타임 타입 검증 및 스키마 정의

**배포 및 인프라**

- **Vercel**: 서버리스 배포 플랫폼으로 자동 배포 및 스케일링
- **pnpm**: 효율적인 패키지 관리

**핵심 구현 기술**

**1. Next.js 기반 풀스택 웹**

- React 19.2 + React Compiler로 자동 성능 최적화
- Next.js 16의 Server Components 및 Server Actions 아키텍처
- after() API를 활용한 비동기 백그라운드 LLM 처리(일기 분석 등)

**2. 디자인 시스템**

- Tailwind CSS v4 기반 커스텀 디자인 시스템 구축
- Figma MCP 통합으로 디자인-코드 동기화
- 체계적인 spacing/typography/color 변수 관리
- 직접 디자인한 커스텀 SVG 아이콘셋 및 Toss Face 스타일 이모지 컴포넌트

**3. AI 워크플로우 구축**

- LangChain + Structured Output Parser로 타입 안전한 AI 응답 처리
- Zod 스키마 기반 런타임 검증
- 컨텍스트 인식 감정 분석 파이프라인

**4. Server Actions 기반 아키텍처**

- Server-side data fetching으로 빠른 초기 로딩 속성 제공
- Next.js Server Actions를 활용한 빠르고 효율적인 서버-클라이언트 통신 구현
- React Hook Form과 통합된 폼 처리
- 자동 캐시 revalidation

**5. 성능 최적화**

- Server-side data fetching 및 Streaming
- Suspense 및 Loading UI를 활용한 점진적 렌더링
- Database indexing 및 쿼리 최적화, Prisma의 ORM을 활용한 스키마 관리

## 향후 개선 혹은 발전 방안

### 1. AI 기능 고도화

- **감정 패턴 분석**: 장기간 감정 데이터를 분석하여 사용자의 감정 패턴 및 트리거 요인 파악
- **예측 모델**: 과거 감정 데이터를 기반으로 미래 감정 상태 예측 및 사전 대응 조언 제공
- **음성 일기**: 음성 입력을 통한 일기 작성 및 음성 분석으로 감정 강도 측정
- **이미지 분석**: 사진 첨부 시 이미지 내용 분석으로 감정 컨텍스트 보강

### 2. 사용자 경험 개선

- **모바일 앱**: React Native 또는 PWA를 활용한 네이티브 모바일 앱 개발
- **푸시 알림**: 일기 작성 리마인더, 긍정적인 메시지 전송
- **다크 모드**: 야간 사용을 위한 다크 테마 지원
- **다국어 지원**: 영어, 일본어 등 다국어 서비스 확장

### 3. 커뮤니티 기능

- **익명 커뮤니티**: 비슷한 감정을 경험하는 사용자들 간의 익명 소통 공간
- **감정 공유**: 일기를 익명으로 공유하고 다른 사용자의 응원 받기
- **전문가 연결**: 심각한 감정 상태 감지 시 전문 심리 상담사와 연결

### 4. 데이터 분석 및 인사이트

- **월간/연간 리포트**: 감정 변화 추이를 담은 주기적 리포트 생성
- **목표 설정 및 추적**: 감정 개선 목표 설정 및 달성 여부 추적
- **감정 트리거 분석**: 특정 감정을 유발하는 상황이나 키워드 분석
- **건강 데이터 연동**: 수면, 운동 등 건강 데이터와 감정 상태 상관관계 분석

### 5. 보안 및 프라이버시

- **엔드투엔드 암호화**: 민감한 일기 및 채팅 데이터의 완전한 암호화
- **로컬 모드**: 인터넷 연결 없이 오프라인에서 일기 작성 및 저장
- **데이터 자동 삭제**: 사용자 설정에 따른 오래된 데이터 자동 삭제 기능

### 6. 비즈니스 모델

- **프리미엄 구독**: 고급 AI 기능, 무제한 채팅, 상세 통계 등 제공
- **기업 B2B 서비스**: 직장인 대상 감정 케어 솔루션 제공
- **심리 상담 기관 연계**: 전문 상담이 필요한 사용자를 위한 파트너십 구축
